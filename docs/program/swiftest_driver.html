Driver program for the Swiftest integrators. Unlike the earlier Swift and Swifter drivers, in Swiftest all integrators are run from this single program.

Adapted from Swifter by David E. Kaufmann's Swifter driver programs swifter_[bs,helio,ra15,rmvs,symba,tu4,whm].f90 Adapted from Hal Levison and Martin Duncan's Swift driver programs Read in the user-defined parameters file and the initial conditions of the system Define the maximum number of threads

Step the system forward in time

Evaluate any discards or collisional outcomes

If the loop counter is at the output cadence value, append the data file with a single frame If the loop counter is at the dump cadence value, dump the state of the system to a file in case it needs to be restarted
