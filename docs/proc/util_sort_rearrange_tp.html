module subroutine util_sort_rearrange_tp(self, ind)

Rearrange Swiftest massive body structure in-place from an index list. This is a helper utility used to make polymorphic sorting work on Swiftest structures.

Type IntentOptional AttributesName

Swiftest test particle object
integer(kind=I4B), intent(in), dimension(:):: ind

Index array used to restructure the body (should contain all 1:n index values in the desired order)
