module subroutine io_param_reader(self, unit, iotype, v_list, iostat, iomsg)

Read in parameters for the integration Currently this procedure does not work in user-defined derived-type input mode e.g. read(unit,'(DT)') param as the newline characters are ignored in the input file when compiled in ifort.

Adapted from David E. Kaufmann's Swifter routine io_init_param.f90 Adapted from Martin Duncan's Swift routine io_init_param.f Do basic sanity checks on the input values
